Tout ce que vous devez savoir sur le ciel étoilé et les LED d'ambiance

Le ciel de toit est déposé délicatement, puis nous perçons de minuscules trous pour insérer les fibres optiques une par une. Les fibres sont reliées à un générateur de lumière LED caché dans le coffre ou sous un siège. L'installation est non invasive et réversible. Le pavillon est ensuite reposé sans aucune trace visible.

Cela dépend de l'effet souhaité : 200-300 fibres pour un effet léger et discret, 400-600 fibres pour un effet réaliste (recommandé), 700-1000 fibres pour un effet "voie lactée" dense et spectaculaire. Nous vous conseillons selon la taille de votre pavillon et le rendu désiré.

Non, la consommation est très faible (environ 5-15W selon le nombre de fibres), équivalent à un plafonnier LED. Le système LED moderne est ultra-économe. Aucun impact sur la batterie, même avec une utilisation prolongée. Vous pouvez laisser le ciel étoilé allumé sans risque.

Oui, les LED d'ambiance sont universelles et compatibles avec tous les véhicules. L'installation s'adapte à votre habitacle : plancher, portes, tableau de bord, sièges. Les bandes LED sont fines et flexibles, permettant de suivre tous les contours. Le câblage est invisible, dissimulé dans les garnitures existantes.

Via télécommande infrarouge (fournie) ou application smartphone selon le kit. Vous pouvez changer la couleur, l'intensité, le mode (fixe, dégradé, pulsation, synchronisation musique). Les réglages sont mémorisés : au démarrage, les LED reprennent automatiquement vos derniers paramètres.
